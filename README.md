# 简介

这是用AI生成的学校闹钟系统，可用于日常学习。

# 使用方式：

## 1.在线预览

[点我](http://alarm.8008088.xyz/)

## 2.下载发布版zip,解压使用

## 3.源码本地运行

### 1.克隆项目到本地
   `git clone https://github.com/Juna3066/study-alarm-system.git`
   
### 2. 安装依赖
   
   **本地电脑安装了:**  Node.js

   项目根目录，运行`npm install`

### 3. 运行项目

   项目根目录，运行`npm run dev`
